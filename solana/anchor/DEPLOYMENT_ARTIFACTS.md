# 📦 Deployment Artifacts

This document explains the structure of deployment artifacts generated by `deploy-programs.sh`.

## Directory Structure

```
solana/anchor/
├── logs/                                           # Deployment logs (gitignored)
│   ├── deployment-localnet-YYYYMMDD-HHMMSS.log
│   ├── deployment-devnet-YYYYMMDD-HHMMSS.log
│   └── deployment-mainnet-YYYYMMDD-HHMMSS.log
├── deployments/                                    # Deployment summaries and symlinks (gitignored)
│   ├── localnet-deployment-YYYYMMDD-HHMMSS.json
│   ├── localnet-latest.json
│   ├── devnet-deployment-YYYYMMDD-HHMMSS.json
│   ├── devnet-latest.json
│   ├── mainnet-deployment-YYYYMMDD-HHMMSS.json
│   └── mainnet-latest.json
├── qwami/                                          # QWAMI program addresses (committed)
│   ├── localnet-addresses.json
│   ├── devnet-addresses.json
│   └── mainnet-addresses.json
└── kwami/                                          # KWAMI program addresses (committed)
    ├── localnet-addresses.json
    ├── devnet-addresses.json
    └── mainnet-addresses.json
```

## File Descriptions

### 1. Deployment Logs (`logs/`)

**Purpose**: Complete console output of each deployment  
**Naming**: `deployment-{cluster}-YYYYMMDD-HHMMSS.log`  
**Content**: Full transcript including:
- All script output
- Command execution
- Errors and warnings
- Timestamps

**Example**:
```
logs/deployment-devnet-20251217-193640.log
logs/deployment-mainnet-20251217-194530.log
```

**Usage**:
```bash
# View latest devnet log
tail -f logs/deployment-devnet-*.log

# View latest mainnet log
tail -f logs/deployment-mainnet-*.log

# Search for errors in specific cluster
grep -i error logs/deployment-devnet-20251217-193640.log

# View all logs for a cluster
ls -lth logs/deployment-devnet-*.log
```

### 2. Deployment Summaries (`deployments/`)

**Purpose**: Structured JSON summary of deployment state  
**Naming**: `{cluster}-deployment-YYYYMMDD-HHMMSS.json`  
**Content**: Key deployment information in JSON format

**Example**:
```json

  "cluster": "devnet",
  "timestamp": "2025-12-17T19:36:40Z",
  "deployer": "3TYRKswBCUy8agGNBF3wpg4AoiahZWKBKJB3ZJhybscf",
  "balance": "4.84554944",
  "programs": {
    "qwami": {
      "programId": "6CAgdgpPq8Np78LsDwREJqFPh9rM5Jh6RSS8eZ37kZuv",
      "explorer": "https://explorer.solana.com/address/6CAgdgpPq8Np78LsDwREJqFPh9rM5Jh6RSS8eZ37kZuv?cluster=devnet",
      "initialized": true

    "kwami": {
      "programId": "DoAJAykwUrSDjraDegK4AJ1GCoztLYrTvKhUJHaFbSsD",
      "explorer": "https://explorer.solana.com/address/DoAJAykwUrSDjraDegK4AJ1GCoztLYrTvKhUJHaFbSsD?cluster=devnet",
      "initialized": true

```

**Usage**:
```bash
# View latest devnet deployment
cat deployments/devnet-latest.json | jq

# View latest mainnet deployment
cat deployments/mainnet-latest.json | jq

# Get program IDs
jq -r '.programs.qwami.programId' deployments/devnet-latest.json
jq -r '.programs.kwami.programId' deployments/devnet-latest.json

# Check initialization status
jq -r '.programs.qwami.initialized' deployments/devnet-latest.json
```

## Git Strategy

### Committed Files
- `qwami/localnet-addresses.json` - Committed (local development state)
- `qwami/devnet-addresses.json` - Committed (devnet deployment state)
- `qwami/mainnet-addresses.json` - Committed (mainnet deployment state)
- `kwami/localnet-addresses.json` - Committed (local development state)
- `kwami/devnet-addresses.json` - Committed (devnet deployment state)
- `kwami/mainnet-addresses.json` - Committed (mainnet deployment state)

### Gitignored Files
- `logs/` - Gitignored (large, temporary, developer-specific)
- `deployments/` - Gitignored (can be regenerated, developer-specific)

**Rationale**: 
- Address files are small, critical for scripts, and represent the canonical deployment state
- Logs can be large and contain developer-specific information
- Deployment summaries can be regenerated from address files if needed

## Querying Deployment State

### Quick Status Check
```bash
# Check if programs are deployed on devnet
ls -lh qwami/devnet-addresses.json kwami/devnet-addresses.json

# Check all deployment states
ls -lh qwami/*-addresses.json kwami/*-addresses.json

# Get program IDs quickly for devnet
grep programId qwami/devnet-addresses.json
grep programId kwami/devnet-addresses.json

# Get program IDs for mainnet
grep programId qwami/mainnet-addresses.json
grep programId kwami/mainnet-addresses.json
```

### Detailed Queries
```bash
# Get all QWAMI PDAs for devnet
jq -r 'to_entries[] | select(.key != "timestamp" and .key != "cluster" and .key != "wallet" and .key != "transaction") | "\(.key): \(.value)"' qwami/devnet-addresses.json

# Get initialization transaction
jq -r '.transaction' qwami/devnet-addresses.json

# Get deployment timestamp
jq -r '.timestamp' qwami/devnet-addresses.json

# Compare program IDs across clusters
echo "Localnet: $(jq -r '.programId' qwami/localnet-addresses.json 2>/dev/null || echo 'Not deployed')"
echo "Devnet:   $(jq -r '.programId' qwami/devnet-addresses.json 2>/dev/null || echo 'Not deployed')"
echo "Mainnet:  $(jq -r '.programId' qwami/mainnet-addresses.json 2>/dev/null || echo 'Not deployed')"
```

### Explorer Links
```bash
# Open QWAMI devnet in browser
PROGRAM_ID=$(jq -r '.programId' qwami/devnet-addresses.json)
echo "https://explorer.solana.com/address/${PROGRAM_ID}?cluster=devnet"

# Open KWAMI mainnet in browser
PROGRAM_ID=$(jq -r '.programId' kwami/mainnet-addresses.json)
echo "https://explorer.solana.com/address/${PROGRAM_ID}"

# Open collection mint on explorer
COLLECTION=$(jq -r '.collectionMint' kwami/devnet-addresses.json)
echo "https://explorer.solana.com/address/${COLLECTION}?cluster=devnet"
```

## Troubleshooting

### Missing Deployment Summary
If `deployments/{cluster}-latest.json` is missing, you can recreate it from the address files:

```bash
# Recreate deployment summary (example for devnet)
CLUSTER="devnet"
cat > deployments/${CLUSTER}-latest.json << 'EOFF'
{
  "cluster": "${CLUSTER}",
  "timestamp": "$(date -u +%Y-%m-%dT%H:%M:%SZ)",
  "programs": {
    "qwami": $(cat qwami/${CLUSTER}-addresses.json | jq '{programId, initialized: true}'),
    "kwami": $(cat kwami/${CLUSTER}-addresses.json | jq '{programId, initialized: true}')
  }
}
EOFF
```

### Viewing Old Deployments
```bash
# List all deployment logs for devnet
ls -lth logs/deployment-devnet-*.log

# List all deployment logs for mainnet
ls -lth logs/deployment-mainnet-*.log

# List all deployment summaries for devnet
ls -lth deployments/devnet-deployment-*.json

# View specific deployment
cat deployments/devnet-deployment-20251217-193640.json | jq

# Compare two deployments
diff <(jq -S . deployments/devnet-deployment-20251217-193640.json) \
     <(jq -S . deployments/devnet-deployment-20251217-201230.json)
```

### Cleaning Up Old Artifacts
```bash
# Keep only last 10 devnet logs
cd logs && ls -t deployment-devnet-*.log | tail -n +11 | xargs rm -f

# Keep only last 10 mainnet logs
cd logs && ls -t deployment-mainnet-*.log | tail -n +11 | xargs rm -f

# Keep only last 5 deployment summaries per cluster
cd deployments && ls -t devnet-deployment-*.json | tail -n +6 | xargs rm -f
cd deployments && ls -t mainnet-deployment-*.json | tail -n +6 | xargs rm -f

# Clean all localnet artifacts (development only)
rm -f logs/deployment-localnet-*.log
rm -f deployments/localnet-deployment-*.json
```

## Integration with Scripts

The deployment artifacts are automatically used by:
- `deploy-programs.sh` - Creates logs, summaries, and references address files
- `initialize-programs.sh` - Reads program IDs from address files
- `qwami/scripts/initialize-qwami.ts` - Creates and updates address file
- `kwami/scripts/initialize-kwami.ts` - Creates and updates address file
- Test scripts - Read addresses from JSON files

All scripts check for the existence of `{program}/{cluster}-addresses.json` to determine deployment state for the specified cluster.

## Multi-Cluster Workflow

### Typical Deployment Sequence

1. **Localnet** (Development)
   ```bash
   ./deploy-programs.sh
   # Select: 1) Localnet
   # Creates: qwami/localnet-addresses.json, kwami/localnet-addresses.json
   ```

2. **Devnet** (Testing)
   ```bash
   ./deploy-programs.sh
   # Select: 2) Devnet
   # Creates: qwami/devnet-addresses.json, kwami/devnet-addresses.json
   ```

3. **Mainnet** (Production)
   ```bash
   ./deploy-programs.sh
   # Select: 3) Mainnet
   # Requires double confirmation
   # Creates: qwami/mainnet-addresses.json, kwami/mainnet-addresses.json
   ```

### Switching Between Clusters

```bash
# Check which clusters are deployed
ls -1 qwami/*-addresses.json kwami/*-addresses.json

# View current deployment for each cluster
for cluster in localnet devnet mainnet; do
  echo "=== $cluster ==="
  jq -r '.programId' qwami/${cluster}-addresses.json 2>/dev/null || echo "Not deployed"
done

# Configure Solana CLI for specific cluster
solana config set --url devnet
solana config set --url mainnet-beta
solana config set --url http://localhost:8899  # localnet
```
