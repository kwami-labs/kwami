# 🚀 Quick Start Guide

## Your Kwami Website is Ready!

The development server is already running at:

**🌐 http://localhost:5173**

Open this URL in your browser to see your beautiful scrolling website!

## What You'll See

### 📱 Layout
- **Left Side**: Scrollable text content explaining the Kwami project
- **Right Side**: Animated blob that morphs as you scroll

### 🎨 Interactive Features
1. **Scroll down** to see different sections (6 total)
2. **Watch the blob** morph into different shapes
3. **See the colors** change progressively
4. **Click the blob** to randomize colors instantly! 🎲

### ✨ The 6 Sections
1. **Meet Kwami** - Introduction (circular blob)
2. **🧠 Mind** - AI features (star-like blob)
3. **🧬 Body** - Visuals (squiggly blob)
4. **✨ Soul** - Personality (pulsing blob)
5. **🎨 Customization** - Options (spiral blob)
6. **🚀 Get Started** - Call to action (heart-like blob)

## 🛠️ Development Commands

```bash
cd web

# Start dev server (already running!)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview

# Stop dev server
# Press Ctrl+C in the terminal
```

## 📝 Making Changes

### Edit Content
Open `index.html` and modify the `.text-section` divs

### Change Colors
Edit `src/main.ts` - look for `colorPalettes` array

### Modify Animations
Edit `src/style.css` - adjust animation durations and effects

### Update Blob Behavior
Edit `src/main.ts` - modify the `KwamiBlob` class

**Changes auto-reload!** The dev server watches for file changes.
