## 📚 API Reference

### Onboarding Tour
```typescript
const tour = (window as any).onboardingTour;
tour.start();    // Start tour
tour.next();     // Next step
tour.previous(); // Previous step
tour.skip();     // Skip tour
tour.finish();   // Complete tour
tour.reset();    // Reset (for testing)
```

### Quick Actions
```typescript
// Automatically opens with Cmd/Ctrl+K
// Or programmatically:
const quickActions = (window as any).quickActions;
quickActions.show();
quickActions.hide();
quickActions.toggle();
```

### Theme Manager
```typescript
const theme = (window as any).themeManager;
theme.setTheme('rose');      // Change theme
theme.nextTheme();           // Cycle themes
theme.getCurrentTheme();      // Get current
theme.showPerformanceIndicator();
```

### Social Features
```typescript
const social = (window as any).socialFeatures;
social.nativeShare();         // Native share
social.showShareModal();      // Show modal
social.getGitHubStars();      // Get stars count
```
