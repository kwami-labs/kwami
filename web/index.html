🤝 What is Kwami?

Kwami is an independent, reusable JavaScript library for creating engaging 3D AI companions.

Combines Three.js visual rendering, Web Audio API, and cutting-edge AI voice technology into one unified package

Framework agnostic - integrates seamlessly with React, Vue, Svelte, Angular, or vanilla JavaScript

06

⚡ How It Works

Built on a Mind-Body-Soul architecture that mirrors human-like intelligence and expression.

Body: WebGL scene with animated blob mesh, audio analysis via FFT, and simplex noise deformation

Mind: Provider-based AI system with hot-swappable backends for TTS, STT, and conversations

Soul: Emotional trait spectrum that generates contextual system prompts for personality-aware AI

07

🎮 Interactive Features

Rich interaction system that responds to user input, audio, and AI states.

Click/touch creates liquid-like ripples using raycasting and vertex displacement with configurable strength and duration

Four animation states: idle (breathing), listening (inward spikes), thinking (chaotic), speaking (outward spikes)
