# KWAMI Documentation

Welcome to the KWAMI ecosystem documentation! This is your central hub for all KWAMI-related documentation.

## 📚 Documentation by Project

The KWAMI ecosystem consists of multiple projects, each with dedicated documentation:

### [1. KWAMI Core Library](./kwami/README.md)
The foundational library powering all KWAMI applications.
- 3D character rendering and animation
- AI-powered behavior and decision-making
- State management and persistence
- Modular architecture

**Key Topics:** Architecture, API Reference, Core Concepts, Mind/Body/Soul systems

---

### [2. Playground (pg)](./pg/README.md)
Interactive development and testing environment.
- Visual blob editor with real-time preview
- ElevenLabs TTS integration
- Audio-reactive animations
- Personality configuration

**Key Topics:** Features, Setup, Usage Guide, Customization

---

### [3. KWAMI App](./app/README.md)
Main Nuxt.js application for the KWAMI experience.
- Interactive 3D visualization
- Voice interaction
- Multi-language support
- Supabase integration

**Key Topics:** Setup, Architecture, Features, Development

---

### [4. Solana Programs](./solana/README.md)
Blockchain integration with KWAMI NFT and QWAMI token.
- KWAMI NFT Program (10B supply by 2100)
- QWAMI Token Program (utility token)
- DNA-based uniqueness validation
- Metaplex standard compliance

**Key Topics:** Token Economics, Supply Schedule, Testing, Programs

---

### [5. Candy Machine](./candy/README.md)
NFT minting interface for KWAMI NFTs.
- Interactive NFT preview
- DNA generation and validation
- Arweave storage integration
- Solana blockchain minting

**Key Topics:** Minting Process, DNA System, Deployment, Environment Setup

---

### [6. NFT Marketplace](./market/README.md)
Trading platform for KWAMI NFTs.
- Browse and search listings
- Buy and sell NFTs
- User profiles
- Real-time marketplace data

**Key Topics:** Features, Setup, Trading, Integration

---

### [7. DAO](./dao/README.md)
Decentralized governance platform.
- Proposal creation and voting
- Treasury management
- Token-weighted governance
- Community decision-making

**Key Topics:** Governance, Voting System, Proposals, Treasury

---

### [8. Website](./web/README.md)
Public-facing website for the KWAMI ecosystem.
- Landing page and feature showcase
- Interactive demo
- Responsive design
- PWA support

**Key Topics:** Design, Features, Deployment, Performance

---
