## 🎯 Overview

The KWAMI NFT Marketplace is a web application for trading KWAMI NFTs on Solana. It provides:

- **NFT Listing** - List your KWAMIs for sale
- **NFT Trading** - Buy and sell KWAMIs
- **Auction System** - Timed auctions for rare KWAMIs
- **Collection Browser** - Browse all available KWAMIs
- **Price Discovery** - Market-driven pricing
- **Wallet Integration** - Seamless Solana wallet connection
- **Transaction History** - Track all marketplace activity

## 🏗️ Tech Stack

- **Framework**: Nuxt 3
- **Blockchain**: Solana
- **NFT Standard**: Metaplex
- **UI**: Vue 3 + Tailwind CSS
- **State Management**: Pinia
- **Wallet**: Solana Wallet Adapter

## 🚀 Quick Start

```bash
cd market
npm install
npm run dev
```
