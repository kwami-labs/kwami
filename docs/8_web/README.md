## 🎯 Overview

The KWAMI website is the main entry point for users to discover and interact with the KWAMI ecosystem. It provides:

- **Landing Page** - Introduction to KWAMI
- **Feature Showcase** - Demonstrate KWAMI capabilities
- **Interactive Demo** - Try KWAMI in the browser
- **Documentation Links** - Access to all documentation
- **Download/Access** - Get started with KWAMI
- **Community Links** - Social media and community resources

## 🏗️ Tech Stack

- **Framework**: Vite + TypeScript
- **3D Rendering**: Three.js
- **UI**: Vanilla JavaScript + CSS
- **Deployment**: Static hosting (Vercel, Netlify, etc.)
- **PWA**: Progressive Web App capabilities

## 🚀 Quick Start

```bash
cd web
npm install
npm run dev
```

The website will be available at `http://localhost:5173`

For detailed setup, see [QUICK_START_PHASE1.md](./QUICK_START_PHASE1.md).
