## 📦 Package Information

The KWAMI core library is the foundation of the KWAMI ecosystem, providing:
- 3D character rendering and animation
- AI-powered behavior and decision-making
- State management and persistence
- Modular architecture for extensions

For implementation details, see the source code in `/src/`.
