# Mind Menu - Quick Reference Card

## 🚀 Quick Start

1. Enter your ElevenLabs API Key
2. Click "Initialize AI Agent"
3. Select a voice or use default (Adam)
4. Enter text to speak
5. Click "🎤 Speak Text"

## 🎚️ Voice Settings Guide

### Stability

- **Low (0.2-0.4):** Expressive, varied
- **Medium (0.5-0.6):** Balanced ⭐
- **High (0.7-0.9):** Consistent, stable

### Similarity Boost

- **Keep at 0.75-0.85** for best results ⭐

### Style

- **Start at 0.0** (stable) ⭐
- Increase cautiously for more expression

### Speaker Boost

- **Keep enabled** ✅

## 🔧 Common Tasks

### Change Voice

1. Select from dropdown or enter custom ID
2. Voice changes immediately

### Test Settings

1. Adjust sliders
2. Click "✅ Apply Voice Settings"
3. Click "👂 Preview Voice"

### Save Configuration

1. Click "💾 Export Configuration"
2. JSON file downloads automatically

### Load Configuration

1. Click "📂 Import Configuration"

## 🎯 Best Practices

### For Quality

- ✅ Use Multilingual v2 model
- ✅ Keep similarity boost at 0.75+
- ✅ Start with Natural preset
- ✅ Enable speaker boost

### For Speed

- ⚡ Use Turbo v2 model
- ⚡ Enable latency optimization
- ⚡ Use PCM format

### For Accuracy

- 🎯 Add custom pronunciations
- 🎯 Use pronunciation dictionary
- 🎯 Test with actual content

---

## 🔍 Troubleshooting

### Voice Sounds Distorted

→ Lower style to 0.0, increase stability to 0.7+

### Too Slow

→ Switch to Turbo v2 model, enable latency optimization

### Wrong Pronunciation

→ Add word to pronunciation dictionary

### Microphone Not Working

→ Check browser permissions, refresh page
