# kwami-ai

3D AI companion library with voice, memory, and tools.

## Development Setup

### Prerequisites

- Node.js 18+
- [pnpm](https://pnpm.io/)

### Installation

```bash
pnpm install
```

### Scripts

```bash
# Build the library
pnpm build

# Watch mode (TypeScript)
pnpm dev

# Run tests
pnpm test

# Lint & format
pnpm lint
pnpm format
```

### Playground

A web playground is available in the `pg/` folder:

```bash
cd pg
pnpm install
pnpm dev
```
